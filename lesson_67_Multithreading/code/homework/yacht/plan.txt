Создание класса Yacht:

Определите необходимые поля класса (manufacturer, date, length, bodyMaterial, price).
Реализуйте конструктор для инициализации полей.
Переопределите методы equals, hashCode, и toString для корректного сравнения и отображения объектов.
Создание класса YachtManager:

Добавьте список (например, List<Yacht> yachts) для хранения яхт.
Реализуйте методы для добавления (addYacht), удаления, и получения яхт.
Реализуйте методы для выполнения задач:
displayAllYachts: отображение всех яхт.
displayYachtsByMaterial(String material): отображение яхт с указанным материалом корпуса.
displayYachtsExpensiveThan(double price): отображение яхт, дороже указанной цены.
displayYachtsByYearRange(int startYear, int endYear): отображение яхт с годом выпуска в заданном диапазоне.
sortYachtsByDescendingOrder(): сортировка яхт в порядке убывания цены.
calculateAverageYachtPrice(): расчет средней стоимости яхт.
Создание класса Main:

Добавьте несколько яхт с использованием метода addYacht.
Вызовите различные методы YachtManager для выполнения задач.
